# Scattered

> **🏆 3rd Place Winner** at the *Comes In Pairs Game Jam* (The Woodlands Secondary School CS Club)  
> *Note: This was our team's first-ever game jam!*

🎮 **[Play the game on Itch.io!](https://cheesyinthehead.itch.io/scattered)**

## About the Game
The cards are scattered between the worlds! *Scattered* is a puzzle-collection game where players must gather missing pairs of cards to win. 

The game begins with a classic memory-match phase. After the cards fall, the player must navigate between two distinct worlds to collect and re-pair all the missing cards.

**Theme Alignment ("Comes in Pairs"):**
The game starts with memory match where the player needs to form **pairs**. After, there are *two* worlds (**pair**), 2D and 3D, where the cards are scattered with the objective being to re-**pair** all the cards.

## Technical Challenges & Learnings
This was the first-ever game jam for both of us, meaning we had to learn Unity and C# game development entirely on the fly during the hacking period. 

We specifically had to figure out:
* **Bridging 2D and 3D Environments:** Managing the transition between the 2D and 3D worlds meant we had to learn two entirely different physics systems, camera perspectives, and movement scripts.
* **Game State Management:** We had to write C# scripts to track the memory match phase, store the player's progress, and ensure the collected cards triggered the correct win conditions across different scenes.
* **Rapid Skill Acquisition:** Going from zero Unity experience to a fully playable, multi-level game under a deadline taught us how to quickly make a MVP and fine tune from there.
